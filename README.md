<h1>🎮 C# e Desenvolvimento de Jogos com Unity</h1>

Repositório destinado ao armazenamento de códigos, exemplos e projetos desenvolvidos durante os estudos de C# e Unity 💻

Contém exercícios para aprendizado da linguagem C#, conceitos fundamentais de programação orientada a objetos e um projeto completo do jogo Flappy Bird desenvolvido na Unity 🚀

##

## 📚 Conteúdo

### 🧠 Fundamentos de C#

- Variáveis
- Operadores aritméticos
- Operadores relacionais
- Operadores lógicos
- Estruturas condicionais
- Laços de repetição
- Arrays
- Listas
- Ciclo de vida de scripts
- Operador ternário
- Controle de fluxo

### 🎮 Projeto Flappy Bird

Jogo inspirado no clássico Flappy Bird, desenvolvido utilizando a Unity Engine.

Recursos implementados:

- 🐦 Controle do personagem
- 🚧 Geração de obstáculos
- 📈 Sistema de pontuação
- 💀 Detecção de colisões
- 🎬 Animações
- 🖼️ Interface de menu
- 🔄 Reinício de partida
- 🎯 Gerenciamento de jogo
